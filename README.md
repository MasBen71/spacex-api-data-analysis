# SpaceX API Data Collection

## Project Overview

This project focuses on collecting and preparing SpaceX launch data using API-based data access.

The goal is to build a structured dataset that can later be used for exploratory data analysis, visualization, and machine learning prediction tasks.

## Problem Statement

Real-world data science projects often begin with data collection from APIs. This project demonstrates how to retrieve launch-related data, inspect its structure, clean relevant fields, and prepare it for analysis.

## Objectives

- Connect to a public SpaceX-related API
- Retrieve launch data
- Convert API responses into a structured data format
- Clean and organize the dataset
- Prepare data for later analysis and machine learning

## Data Science Workflow

1. API request
2. JSON response inspection
3. Data extraction
4. Data cleaning
5. DataFrame creation
6. Export or prepare data for downstream analysis

## Tools and Libraries

## Key Skills Demonstrated

- API data collection
- JSON handling
- Data wrangling
- Dataset construction

- Preparing real-world data for analysis

## Possible Use Cases

The prepared dataset can be used for:

- Launch success analysis
- Launch site comparison
- Payload mass analysis
- Orbit type analysis
- Machine learning prediction
- Dashboard creation
